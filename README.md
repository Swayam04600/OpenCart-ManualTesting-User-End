## 1. Project Overview

This project demonstrates end-to-end **Manual Testing** performed on an OpenCart based E-commerce web application.

OpenCart is an online shopping platform where users can:

* Register an account
* Browse products
* Add items to cart
* Place orders
* Manage order history

The objective of this project was to validate the system functionality, identify defects, and evaluate whether the application is ready for production release.

## 8. Test Execution Summary

All planned test cases were executed and results recorded in the Test Execution Report.

Based on observed behavior, the application contains functional and validation defects that should be fixed before production deployment.

---

## 9. Conclusion

The OpenCart application is **partially stable** but not fully production-ready due to issues in checkout validation and error handling.

Further regression testing is recommended after defect fixes.
